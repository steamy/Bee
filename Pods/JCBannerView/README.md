## Installation

Installation with CocoaPods

```
pod "JCBannerView", "1.0.1"
```

Installation with Carthage

```
github "lijingcheng/JCBannerView" ~> 1.0.1
```
need dependency SDWebImage(v3.8.2) and Masonry(v1.0.2)
